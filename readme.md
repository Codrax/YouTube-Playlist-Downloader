# Python YouTube Music Playlist Downloader

## Dependencies
### PIP:
- mutagen
- prompt_toolkit

### Packages:
- chafa

### Binaries
- ffmpeg
- ffplay
- ffprobe

## Installation (Arch)
```
sudo pacman -Sy chafa python-mutagen python-prompt_toolkit
```

You also **must** download the 3 ffmpeg binaries and place them in the same directory as the script.
